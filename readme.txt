Redirect you to the Random post of your blog from a post or a page.

== Description ==

Want some fun in blog?

Like the "I'm Feeling Lucky" Function in Google?

Install this plugin now!

This plugin can Redirect you to the Random post of your blog from a post or a page.

And it is just like "I'm Feeling Lucky"!

= How to use this plugin? =

1. First install this plugin.

2. Go in to the post/page you want to redirect's Edit Page.

3. Find Meta Box name "Custom Field".

4. Click "Enter new".

5. Enter "random" in Name Table and enter anything in Value Table.

6. Visit this post/page, now you will be redirect to a random post of your blog!

Live Demo: [Click Here](http://www.arefly.com/try-lucky/) (You will go to a random post of my blog)

== Installation ==

###Updgrading From A Previous Version###

To upgrade from a previous version of this plugin, delete the entire folder and files from the previous version of the plugin and then follow the installation instructions below.

###Installing The Plugin###

Extract all files from the ZIP file, making sure to keep the file structure intact, and then upload it to `/wp-content/plugins/`.

Then just visit your admin area and activate the plugin.
